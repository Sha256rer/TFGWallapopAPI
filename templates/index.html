Rutas de la API Flask

Ruta Método Descripción
/Users/Updates GET Obtiene usuarios con búsquedas a notificar según su frecuencia.
/Users/Updates/<busquedausuario_id> GET Obtiene productos nuevos desde la última notificación de una búsqueda usuario específica.
/Testcon GET Comprueba la conexión con la base de datos.
/Busquedas GET Devuelve todas las búsquedas existentes.
/Busquedas/Update/All GET Actualiza todas las búsquedas que necesitan ser actualizadas según su frecuencia.
/Users/Busquedas/Update/<busqueda_p> POST Actualiza productos de una búsqueda específica y devuelve nuevos productos.
/Users GET Devuelve todos los usuarios.
/Users/<p_name> GET Devuelve el ID del usuario dado su nombre.
/Busquedas/<busqueda_p> GET Devuelve todos los productos asociados a una búsqueda.
/Users/Busquedas/<user_p> GET Devuelve todas las búsquedas asociadas a un usuario.
/Buscar/<producto_p> POST Realiza una búsqueda de producto para un usuario con frecuencia y orden especificados.
/ GET Página principal que renderiza el template index.html.
